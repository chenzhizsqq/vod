# 上传检验<a name="ZH-CN_TOPIC_0161183689"></a>

## 功能描述<a name="section741313354348"></a>

## 请求URI<a name="zh-cn_topic_0128109922_zh-cn_topic_0127940850_section5627230172812"></a>
